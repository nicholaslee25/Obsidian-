## Overview

Keyboards are a full hobby — switches, layouts, cases, keycaps, sound, feel. Deep rabbit hole with a lot of overlap with DIY, electronics, and craft. Also a reliable gift category.

---

## Key Concepts

**Layouts**
- Full (100%) → TKL (87%) → 75% → 65% → 60% → smaller
- Smaller = less desk space, more portability, fewer keys (often remapped via layers)

**Switches**
- **Linear** — smooth all the way down (e.g. Reds, Yellows) — quiet, fast
- **Tactile** — bump without click (e.g. Browns, Boba U4) — feedback without noise
- **Clicky** — bump + click (e.g. Blues, Greens) — satisfying, annoying to others

**Case materials**
- Plastic, aluminum, brass, polycarbonate, wood — each changes sound and feel significantly
- Gasket mount vs. top mount vs. tray mount — affects flex and typing feel

**Keycaps**
- Profiles: Cherry, OEM, SA, MT3, DSA — each has a different height/shape
- Material: PBT (durable, thocky) vs. ABS (smoother, shiny over time)

## PCB

The circuit board inside the keyboard. Determines which switches are compatible, what features are available (RGB, hot-swap, etc.)

- **Hot-swap** — switch sockets that let you pull switches without soldering. Highly recommended for experimenting.
- **Solderable** — solder your switches in place. More permanent, slightly better connection.
- **South vs. north facing LEDs** — matters for RGB shinethrough on certain keycap profiles (Cherry profile conflicts with north-facing)

## Modding Techniques

| Mod | What it does |
|---|---|
| Lube switches | Smoother, quieter keystrokes |
| Lube stabilizers | Removes rattle on larger keys (spacebar, shift, enter) |
| Band-aid mod | Foam pad under stabilizer plate = less clack |
| PE foam mod | Adds a layer of foam between PCB and case = deeper, softer sound |
| Tape mod | Tape strips on PCB back = "poppy" sound signature, free |
| Tempest mod | Thin foam/gaskets between switch pins and PCB = tighter feel |
| Holee mod | O-ring around stabilizer wire = less rattle |

## External Peripherals & Add-ons

- **Stream Deck** — macro pad with LCD buttons; programmable shortcuts, scene switches, macros
- **Shortcut buttons / macro pad** — numpad-style additional keys, remappable
- **Trackpad** — replaces mouse for compact setups; many QMK keyboards support one inline
- **Rotary encoder / knob** — volume, scroll, zoom — common on 75% and custom builds
- **Display** — small OLED or LCD on the board showing WPM, layer, system info
- **Numpad** — separate 17-key block; some layouts include it, others leave it off for desk space

## Layouts — More Detail

| Term | Meaning |
|---|---|
| **TKL (Tenkeyless)** | 87-key layout — removes the numpad. Saves desk space, keeps arrow keys and nav cluster. Most popular non-full-size layout. |
| **75%** | Similar to TKL but tighter — nav cluster compressed, function row kept. |
| **65%** | Arrow keys kept, no function row, no nav cluster. Common enthusiast choice. |
| **60%** | No function row, no arrows, no nav. Compact. Uses layers heavily. |

## Display & Input Add-ons (on keyboards)

| Feature | What it is |
|---|---|
| **TFT screen** | Small LCD/TFT display embedded in the keyboard. Shows WPM, active layer, system stats, animations, art. Mostly cosmetic but cool. |
| **Rotary encoder / knob** | Programmable scroll wheel on the keyboard. Common uses: volume, scroll, zoom, media seek. |
| **Scroll wheel** | Variant of encoder — infinite scroll, typically for media or zoom control. |

## QMK and VIA

**QMK (Quantum Mechanical Keyboard):** Open-source firmware for keyboards. Full programmability — remap every key, create layers, add macros, control RGB, define encoder behavior. Requires flashing the keyboard.

**VIA:** A GUI front-end for QMK-compatible boards. Remap keys visually without writing any code or reflashing. Plug in the keyboard, open VIA in browser, drag and drop key assignments. Much easier entry point than raw QMK.

**QMK/VIA compatible** = the keyboard supports this stack. Highly desirable for any custom or modded build.

| Board | What it is |
|---|---|
| **Zuoya GMK87** | Budget TKL (87-key), hot-swap, gasket mount. Good build quality for the price. One of the best-value TKLs. |
