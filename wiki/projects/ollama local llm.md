## Overview

Download and set up [Ollama](https://ollama.com) — a lightweight local LLM runner. Lets you run open-source models (Llama, Mistral, Gemma, etc.) entirely on your machine, no API calls, no cost, no internet required after download.

**Use case:** TBD

## What It Is

- Runs LLMs locally via a simple CLI (`ollama run llama3`)
- Pulls models from the Ollama library (like Docker images but for models)
- Exposes a local REST API — easy to integrate into scripts or apps
- Low friction: one installer, then `ollama pull <model>`

## Possible Use Cases (to narrow down)

- Local AI for the MCOC autofarm (decision logic without API calls)
- Offline coding assistant
- Private/local chatbot
- Backend for the Discord wiki bot (no API cost)
- Experimentation / tinkering with models
