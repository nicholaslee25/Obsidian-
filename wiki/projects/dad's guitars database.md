**Type:** Project

## Overview

Build a database cataloguing dad's guitar collection — make, model, year, condition, history, any personal notes.

## Fields to Track

| Field | Notes |
|---|---|
| Make / Brand | Gibson, Fender, Yamaha, etc. |
| Model | Stratocaster, Les Paul, etc. |
| Year | Manufacture year if known |
| Serial number | For authentication/value |
| Condition | Excellent / Good / Fair / Needs repair |
| Finish / Color | |
| Pickups | Single coil, humbucker, etc. |
| Modifications | Any non-stock changes |
| Acquisition story | How/when dad got it |
| Sentimental value | Notes on meaning |
| Estimated value | Optional |

## Format Options

- Simple spreadsheet (Google Sheets)
- Notion database
- Custom app (if you want to build it)
